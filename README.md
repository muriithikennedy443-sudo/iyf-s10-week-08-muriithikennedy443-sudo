## Project Description
A React-based community platform called CommunityHub where users can create posts, like posts, delete posts, and search through content. Built as part of IYF Academy Week 08 assignments covering React fundamentals.

## Technologies Used
- React 18
- Vite
- JavaScript (ES6+)
- JSX
- CSS3
- Git & GitHub

## Features
- Create new posts with title and content
- Like/unlike posts with live like counter
- Delete posts
- Search/filter posts by title
- Sidebar with popular posts and tags
- Responsive layout with Header, Main, Sidebar and Footer
- Component composition with reusable shared components

## How to Run
1. Clone this repository
2. Run `npm install`
3. Run `npm run dev`
4. Open browser at `http://localhost:5173`

## Lessons Learned
- How to set up React with Vite
- JSX syntax and expressions
- Creating and composing React components
- Passing data with props
- Managing state with useState hook
- Handling events in React
- Lifting state up between components
- Working with arrays and objects in state

## Challenges Faced
- Understanding when to lift state up to parent components
- Managing multiple state variables across components
- Keeping components reusable while passing the right props
